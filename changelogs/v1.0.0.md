# v1.0.0 Release Notes

> 发布日期：2026-03-18

## 🎉 首次发布

### 核心功能

- **ASN.1 解析**：支持 Hex / Base64 / PEM 格式输入，树形结构展示解析结果，支持自定义 ASN.1 模块（JAR 包扩展）、解析历史记录
- **JSON 工具**：JSON 格式化、压缩、语法高亮，多标签页并行操作
- **编解码工具**：
  - Base64 / URL / Hex 编解码
  - AES（ECB / CBC）对称加解密
  - RSA 公私钥加解密
  - Hash（MD5 / SHA 系列）/ HMAC 计算
- **时间戳转换**：Unix 时间戳与可读时间互转，支持秒 / 毫秒
- **文本对比**：多标签页 Diff 对比，高亮差异行
- **二维码**：二维码生成与图片识别
- **证书解析**：X.509 证书解析，支持 PEM / DER / Base64 / Hex 输入，展示主体、颁发者、有效期、指纹、公钥信息
- **剪贴板历史**：自动记录文本 / 图片 / 文件复制记录，支持收藏、标签、搜索、分页、导出
- **待办清单**：轻量级待办管理，支持提醒时间设置与系统通知推送

### 应用特性

- 悬浮球快捷入口，一键跳转功能模块并自动粘贴剪贴板内容
- 深色 / 浅色 / 跟随系统三种主题，实时切换
- 最小化到系统托盘，支持开机自启动
- 自定义窗口大小与位置记忆
- 多标签页支持（ASN.1 / JSON / 对比 / 证书各自支持同时开多个标签）

### 支持平台

- macOS（Apple Silicon，arm64）
- Windows（x64，NSIS 安装包）
